IMPORTANT

Changes to logging in to Bank@Net online banking

If you also use the mBank@Net mobile bank, you'll now log in to the online bank with a one-time password from the mBank@Net app. Logging in with an SMS token is no longer available for these users. This way is safer and simpler, since you create the one-time password directly in the mobile bank.

Do you use the mBank@Net mobile bank?

To log in you need:

  1. your username,
  2. a one-time password you create in the mBank@Net app.

Create the one-time password like this:

  1. open the mBank@Net app on your phone,
  2. on the app's login screen choose One-time password,
  3. enter the displayed one-time password into Bank@Net.
Don't use the mBank@Net mobile bank?

If you only use the Bank@Net online bank, you can still log in with an SMS token. On the login page choose Log in with SMS OTP and enter your username and password as usual.
